## 项目简介

### 概述

TalkBoard（byv-whiteboard）是一个功能强大的白板录制应用，专为制作教学视频、演讲录制、在线课程等场景设计。它基于 Excalidraw 构建，提供了完整的白板绘图功能，同时集成了屏幕录制、提词器、摄像头叠加等实用功能。

### 适用场景

| 场景 | 描述 |
|------|------|
| 📚 在线教学 | 录制课程内容，配合提词器讲解知识点 |
| 🎤 演讲录制 | 制作演讲视频，支持多演讲页切换 |
| 📝 会议记录 | 记录会议讨论内容，录制演示过程 |
| 🎨 创意展示 | 展示绘图过程，录制创作演示 |

### 核心优势

- **开箱即用**：无需复杂配置，启动即可使用
- **一站式解决方案**：白板 + 录制 + 提词器，无需多个工具配合
- **高质量输出**：自动转换为 H264 编码的 MP4 格式
- **灵活配置**：支持多种画面比例、摄像头位置、麦克风选择

### 1. 白板绘图

基于 Excalidraw 提供的专业白板功能：

| 工具类型 | 包含工具 | 说明 |
|----------|----------|------|
| 绘图工具 | 矩形、圆形、菱形、箭头、线条 | 支持填充、描边、圆角等属性 |
| 书写工具 | 手绘笔、钢笔 | 支持压感、不同粗细 |
| 文本工具 | 文本框 | 支持字体大小、颜色设置 |
| 编辑工具 | 选择、擦除 | 支持多选、组合、锁定 |
| 辅助工具 | 图片、激光笔 | 支持拖拽插入图片 |

**主题支持**：
- 浅色主题（默认）
- 深色主题

### 2. 屏幕录制

专业的屏幕录制功能：

```
录制流程图：

  ┌──────┐    ┌──────────┐    ┌────────┐    ┌──────────┐
  │ 空闲  │───▶│ 区域选择  │───▶│ 准备就绪│───▶│ 正在录制  │
  │ idle │    │ selecting│    │ ready  │    │ recording│
  └──────┘    └──────────┘    └────────┘    └──────────┘
      ▲                                            │
      └────────────────────────────────────────────┘
                      停止录制，下载视频
```

**录制特性**：
- 可拖拽调整录制区域
- 支持固定宽高比（16:9、4:3、9:16 等）
- 实时显示录制时长
- 录制时锁定视图，防止误操作
- 自动转换为 MP4 格式下载

### 3. 演讲页管理

多演讲页支持：

- 添加/删除演讲页
- 拖拽排序
- 键盘快捷键翻页
- 平滑翻页动画
- 根据画面比例自动计算演讲页尺寸

### 4. 提词器

可拖拽的提词器面板：

- 自由拖拽定位（位置自动保存）
- 滚动播放控制
- 滚动速度调节（1-10 级）
- 透明度调节（0-100%）
- 播放/暂停控制

### 5. 摄像头叠加

录制时叠加摄像头画面：

| 设置项 | 选项 |
|--------|------|
| 形状 | 圆形、方形 |
| 位置 | 左上、右上、左下、右下 |
| 大小 | 可调节 |
| 设备 | 支持多摄像头切换 |

### 6. 麦克风录音

同步录制音频：

- 支持多麦克风设备选择
- 实时音量指示
- 与视频同步合成

| 软件 | 版本要求 |
|------|----------|
| Node.js | 18.0+ |
| npm | 9.0+ |

### 安装步骤

```bash
# 1. 克隆仓库
git clone https://github.com/ludengke95/TalkBoard.git

# 2. 进入项目目录
cd TalkBoard

# 3. 安装依赖
npm install

# 4. 启动开发服务器
npm run dev
```

启动后，在浏览器中打开：**http://localhost:3000**

#### 基本操作

1. **选择工具**：点击或按 `V` 键
   - 单击选择元素
   - 拖拽框选多个元素
   - 拖拽移动元素

2. **绘图工具**：点击图标或使用快捷键
   - 矩形：`R`
   - 圆形：`O`
   - 箭头：`A`
   - 线条：`L`
   - 画笔：`D`
   - 文本：`T`

3. **主题切换**：
   - 点击右上角设置图标
   - 选择浅色/深色主题

#### 高级功能

- **组合元素**：选中多个元素后点击组合按钮
- **锁定元素**：选中元素后点击锁定按钮
- **复制粘贴**：`Ctrl+C` / `Ctrl+V`
- **撤销/重做**：`Ctrl+Z` / `Ctrl+Shift+Z`

#### 区域选择操作

| 操作 | 方法 |
|------|------|
| 调整大小 | 拖拽四角的控制点 |
| 移动区域 | 拖拽区域中心 |
| 确认区域 | 点击"确认区域"按钮 |
| 取消选择 | 点击"取消"按钮 |

#### 录制状态说明

| 状态 | 图标 | 说明 |
|------|------|------|
| 空闲 | 🔴 | 可以开始选择录制区域 |
| 区域选择 | ⬜ | 正在拖拽选择录制区域 |
| 准备就绪 | ✅ | 区域已确认，可以开始录制 |
| 正在录制 | ⏺ | 正在录制，显示时长 |

#### 录制注意事项

1. **视图锁定**：录制时视图会被锁定，防止误操作
2. **时长显示**：实时显示录制时长
   - 低于 1 小时：`MM:SS` 格式
   - 超过 1 小时：`HH:MM:SS` 格式
3. **视频格式**：自动转换为 H264 编码的 MP4 格式
4. **文件命名**：`recording_YYYY-MM-DD.mp4`

## 常见问题

### Q1: 录制没有声音？

**解决方案**：
1. 检查设置中麦克风是否启用
2. 确认选择了正确的麦克风设备
3. 检查浏览器是否有麦克风权限
4. 查看音量指示器是否有波动

### Q2: 摄像头无法显示？

**解决方案**：
1. 检查设置中摄像头是否启用
2. 确认选择了正确的摄像头设备
3. 检查浏览器是否有摄像头权限
4. 尝试刷新页面重新获取设备

### Q3: 录制区域选择不准确？

**解决方案**：
1. 使用四角控制点精确调整
2. 选择固定的画面比例（如 16:9）
3. 拖拽中心移动整体位置

### Q4: 视频下载失败？

**解决方案**：
1. 检查浏览器下载设置
2. 确认磁盘空间充足
3. 尝试使用其他浏览器

### Q5: 提词器位置重置了？

**解决方案**：
- 提词器位置保存在 localStorage
- 清除浏览器数据会导致位置重置
- 重新拖拽到合适位置即可

### Q6: 白板内容丢失？

**解决方案**：
- Excalidraw 自动保存到浏览器存储
- 清除浏览器数据会导致内容丢失
- 建议定期导出 .excalidraw 文件备份

### Q7: 支持哪些浏览器？

**推荐浏览器**：
- Google Chrome（推荐）
- Microsoft Edge
- Firefox
- Safari（部分功能可能受限）

### Q8: 如何导出白板内容？

**方法**：
1. 使用 Excalidraw 的导出功能
2. 导出为 PNG/SVG 图片
3. 保存为 .excalidraw 文件
