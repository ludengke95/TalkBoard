# 中文手写字体说明

## 字体信息

- **字体名称**: 杨任东朱石体 Light
- **字体文件**: YangRenDongZhuShiTi-Light-2.ttf
- **文件大小**: ~4.3MB
- **来源**: https://github.com/korbinzhao/excalidraw-cn

## 授权说明

本字体为免费商用字体，可用于商业和非商业项目。

## 用途

用于 Excalidraw 白板应用的中文手写字体显示，提供与英文 Virgil 字体风格一致的中文手写效果。

## 技术实现

通过 CSS `@font-face` 和 `unicode-range` 属性，将本字体作为 Virgil 字体的中文扩展：
- 中文字符使用杨任东朱石体
- 英文字符继续使用 Excalidraw 默认的 Virgil 字体
